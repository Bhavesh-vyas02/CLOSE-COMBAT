# Close Combat - Game Executable

## About
Close Combat is a 2D fighting game built with Python and Pygame, now compiled into a standalone executable.

## How to Run
Simply double-click on `Close-Combat.exe` to start the game. No installation required!

## Game Features
- **6 Unique Characters**: Warrior, Wizard, Huntress, King, Hero Knight, Martial Hero
- **8 Battle Arenas**: Desert Ruins, Mystical Forest, Frozen Arena, Sacred Temple, Peaceful Plains, Volcanic Fury, Murky Swamplands, Grand Arena
- **Game Modes**: Player vs Player (PvP) and Player vs Computer (PvC) with adaptive AI
- **Victory Menu**: Options to replay, change characters, or return to main menu
- **Pause System**: Full pause functionality with sound controls
- **Round System**: Best of 3 rounds with timer
- **Dynamic AI**: Computer opponent adjusts difficulty based on performance

## Controls
### Player 1:
- **A/D**: Move left/right
- **W**: Jump
- **R**: Attack 1
- **T**: Attack 2

### Player 2 (PvP Mode):
- **Arrow Keys**: Move
- **Up Arrow**: Jump
- **Num 1**: Attack 1
- **Num 2**: Attack 2

### General:
- **ESC**: Pause game / Return to menu
- **Mouse**: Navigate menus

## System Requirements
- Windows 10 or later
- ~30MB free disk space
- No additional software installation required (all dependencies are bundled)

## Technical Details
- **File Size**: ~28MB
- **Icon**: Custom icon created from Arena map image
- **Assets**: All game assets (images, fonts, sounds) are bundled inside the executable
- **Dependencies**: Pygame and all required libraries are included

## Troubleshooting
If the game doesn't start:
1. Make sure you have Windows 10 or later
2. Try running as administrator
3. Check that your antivirus isn't blocking the executable
4. Ensure you have enough free disk space

## Victory Menu Features
After winning a match, you can:
- **Replay**: Start a new match with the same characters
- **Change Character**: Go back to character selection
- **Main Menu**: Return to the main menu
- **Quit**: Exit the game
